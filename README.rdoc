== README

Squillo is a websockets-powered chat application for connecting users who want to learn each others' languages. The idea is simple: a user signs up, chooses their native language(s) and the languages they want to practice, see who they've matched with, and start chatting instantly.

Built with:

* Ruby 2.1.5

* Rails 4.2.4

* Private_Pub gem (allows you to publish/subscribe to private channels over Faye server)


<tt>rake doc:app</tt>.
